# React Task 7: Styled Components Task Tracker

A frontend application built to demonstrate the core fundamentals of React, specifically focusing on CSS-in-JS styling using the `styled-components` library.

## 💡 Concepts Covered

- **Styled Components:** Writing CSS directly inside JavaScript to create custom, reusable, and self-styling components (e.g., `<Card>`, `<ActionButton>`).
- **Dynamic Styling:** Passing React Props (like `$isCompleted`) directly into the CSS to automatically change colors based on the component's state.
- **JSX & Functional Components:** Breaking the interface down into reusable pieces.
- **State Management:** Using the `useState` hook to manage the task's completion status.

## 🚀 How to Run Locally

If you want to run this project on your own machine, follow these steps:

1. Clone this repository to your local machine.
2. Open your terminal and navigate to the project folder.
3. Run `npm install` to download the required dependencies (including `styled-components`).
4. Run `npm run dev` to start the local development server.
