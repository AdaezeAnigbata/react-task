# Intro to React: Task Tracker

A simple frontend application built to demonstrate the core fundamentals of React as part of my learning curriculum.

## 💡 Concepts Covered

- **JSX:** Writing UI structures directly within JavaScript.
- **Functional Components:** Breaking the interface down into reusable pieces.
- **Props:** Passing data dynamically from parent to child components.
- **State:** Using the `useState` hook to manage and update the task's completion status.
- **Standard CSS:** Keeping styling clean and separate from the logic.

## 🚀 How to Run Locally

If you want to run this project on your own machine, follow these steps:

1. Clone this repository to your local machine.
2. Open your terminal and navigate to the project folder.
3. Run `npm install` to download the required React dependencies.
4. Run `npm run dev` to start the local development server.
